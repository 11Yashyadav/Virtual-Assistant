# AI Assistant

## Overview
This AI assistant is built using HTML, CSS, and JavaScript. It allows users to:
- Open applications using voice commands
- Search for information online with spoken queries

## Features
- **Voice Recognition**: Uses the Web Speech API to process voice commands.
- **App Launcher**: Opens installed applications using spoken instructions.
- **Web Search**: Fetches search results dynamically based on user queries.
- **User-Friendly UI**: Designed with HTML and CSS for a seamless experience.
- **Lightweight & Fast**: Optimized JavaScript for efficient performance.

## Installation
To set up the AI assistant, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo-name.git
   ```
2. Open the project folder.
3. Launch `index.html` in your preferred browser.

## Usage
To use the assistant, simply speak commands such as:
- **"Open [app name]"** – Launches the specified application.
- **"Search [query]"** – Performs a web search using the given query.
- **"Help"** – Provides guidance on available commands.

You can also customize behavior by modifying the `config.js` file.
